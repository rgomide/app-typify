# Projeto Mobile

## Executar o projeto

```bash
npm install
npm run web
```

## Integração com o backend

A integração com o backend é feita através de uma API REST. Foi criado um hook para facilitar a comunicação com o backend. O código desse arquivo está em [./src/hooks/useServer.js](./src/hooks/useServer.js). Para mais detalhes, consulte o projeto [backend](../backend/).
